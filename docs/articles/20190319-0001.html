config.assets の値を修正したら rails tmp:clear#

事情があって Assets を /public/assets へ生成できなかったので、Changing the assets Path の通りに Rails.application.config.assets.prefix の値を変更したけどパスが変わらなかった。

と思ってたら、rails tmp:clear したら反映された。

先生がこれに気づくまで三時間掛かりました。
